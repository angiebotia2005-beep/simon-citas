# Sistema de Gestión de Citas Médicas

¡Bienvenido al repositorio del **Sistema de Gestión de Citas Médicas**! Este proyecto es una aplicación web completa diseñada para administrar pacientes, especialistas, horarios disponibles y asignación de citas.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React (JavaScript, CSS).
* **Backend:** Spring Boot (Java, Maven).
* **Base de Datos:** MySQL.

## 📁 Estructura del Proyecto

El proyecto está organizado en las siguientes carpetas principales:

* **`/Frontend`**: Contiene la interfaz de usuario, los componentes de React, los estilos y la lógica del cliente.
* **`/Backend`**: Contiene la API REST desarrollada en Spring Boot, los controladores, entidades y la conexión a la base de datos.
* **`/BaseDato`**: Contiene los scripts SQL necesarios para crear la base de datos y agregar datos de prueba iniciales.
